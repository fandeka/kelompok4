Funtastic Four / Online Shop App sistem perdagangan mobile yang dijalankan di bawah Android platform yang digunakan untuk promosi dan penjualan produk Anda dengan aplikasi tunggal. Dengan kuat Admin Panel dapat mengatur urutan, membuat menu kategori dan produk, juga dapat konfigurasi seperti mata uang, pajak, pengguna, dll dari admin panel di web. Anda dapat menambahkan, update, memperbarui atau mengubah itu menu produk, kategori, pajak, mata uang dan perubahan password admin dengan menghasilkan password, dll. Aplikasi ini dibuat oleh Android untuk sisi klien dan kemudian PHP MySQL untuk Admin sisi. Berjalan di bawah Platform Android yang merupakan sistem operasi yang paling populer di dunia. Menggunakan aplikasi ini Anda dapat menyimpan uang dan waktu Anda dalam membuat permohonan perdagangan atau toko online aplikasi.
